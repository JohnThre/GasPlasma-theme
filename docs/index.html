Gas Plasma

An orange-dominant dark theme inspired by ionized gas glow. A single warm, high-contrast palette for Apple Terminal, iTerm2, VS Code, Open VSX-compatible editors, and Zed.

Releases Open VSX

Install Targets

Apple Terminal

Download GasPlasma.terminal from GitHub Releases and open it in Terminal.

open GasPlasma.terminal

iTerm2

Import GasPlasma.itermcolors from iTerm2 color presets.

GasPlasma.itermcolors

VS Code

Install the published VSIX from Open VSX-compatible editors.

jpfchang.gas-plasma-theme

Zed

Install the color theme and file icon theme from the Zed extensions store after marketplace review.

Gas Plasma Theme + Icons

VS Code Icons

File icons are generated from recognizable upstream language and ecosystem logo geometry where available. Product icons are generated from VS Code Codicons to preserve the original editor UI language.

Build Architecture

The release package is generated from one palette, one icon manifest, upstream icon sources, and deterministic build scripts.
