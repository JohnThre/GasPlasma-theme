Gas Plasma

An orange-dominant dark theme inspired by ionized gas glow. A single warm, high-contrast palette for Apple Terminal, iTerm2, VS Code, and Open VSX-compatible editors.

Releases Open VSX

Install Targets

Apple Terminal

Download GasPlasma.terminal from GitHub Releases and open it in Terminal.

open GasPlasma.terminal

iTerm2

Import GasPlasma.itermcolors from iTerm2 color presets.

GasPlasma.itermcolors

VS Code

Install the published VSIX from Open VSX-compatible editors.

johnthre.gas-plasma-theme
